# @unction/prepend

![Tests][BADGE_TRAVIS]
![Stability][BADGE_STABILITY]
![Dependencies][BADGE_DEPENDENCY]

> A => Array<A | B> | string => Array<A | B> | string

Takes a value and puts it at the beginning of the given list.


``` javascript
prepend(4)([5]) // => [4, 5]
prepend("c")("ab") // => "cab"
```
